# Hola Mundo Interactivo

Proyecto estatico listo para publicar en GitHub Pages.

## Estructura

- `index.html`
- `styles.css`
- `script.js`
- `.github/workflows/deploy-pages.yml`

## Publicacion en GitHub Pages

1. Sube el proyecto a un repositorio en GitHub.
2. Asegurate de trabajar sobre la rama `main`.
3. Haz push de estos archivos. El workflow se ejecutara automaticamente.
4. En GitHub, ve a `Settings > Pages` y en `Build and deployment` selecciona `GitHub Actions`.
5. Cuando el workflow termine, tu sitio quedara publicado en:

`https://TU_USUARIO.github.io/TU_REPO/`

Si el repositorio se llama `TU_USUARIO.github.io`, se publicara en la raiz del dominio:

`https://TU_USUARIO.github.io/`
